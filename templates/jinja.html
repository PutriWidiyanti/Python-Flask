statement dan logika

Dalam Python, khususnya saat menggunakan Flask dengan template engine Jinja2 tanda {{ '{{' }}% %{{ '}}' }} adalah sintaks untuk menulis statement (pernyataan) atau logika di dalam file template HTML, digunakan untuk menjalankan kode Python dalam file HTML

Menggunakan Template Inheritance
  • {{ '{{' }}% extends 'base.html' %{{ '}}' }}
    Menandakan bahwa file ini menggunakan template base.html.
  • {{ '{{' }}% block title %{{ '}}' }}
    Mendefinisikan konten khusus untuk blok title di base.html.
  • {{ '{{' }}% block content %{{ '}}' }}
    Mendefinisikan konten khusus untuk blok content.
